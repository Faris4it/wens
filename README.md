# وِنس (Wens) — نموذج أولي (Prototype)

نموذج واجهة أمامية (MVP UI) لمنصة عربية تتيح للمستخدمين الوصول إلى ألعاب رقمية بأسعار مناسبة. هذا مشروع **نموذج تصميم فقط** بدون أي منطق أعمال حقيقي (لا دفع، لا حسابات، لا قاعدة بيانات).

## المعاينة محليًا

المشروع ثابت بالكامل (HTML/CSS/JS فقط)، يكفي فتح `index.html` في المتصفح، أو تشغيل خادم ثابت بسيط:

```bash
npx serve .
# أو
python -m http.server 5500
```

## البنية

```
/
├── index.html            صفحة الموقع الرئيسية
├── css/
│   ├── variables.css      متغيرات التصميم (ألوان، مسافات، خطوط)
│   ├── style.css          التخطيط والمكوّنات
│   └── animations.css     الحركات والانتقالات (keyframes)
├── js/
│   ├── data.js             بيانات وهمية للألعاب والتقييمات والأسئلة الشائعة
│   ├── animations.js       تأثير الكشف عند التمرير + تأثير الموجة على الأزرار
│   └── app.js               التنقل، البحث والفلترة، الأكورديون، النافذة المنبثقة
└── assets/
    ├── images/              أغلفة الألعاب والرسم التوضيحي (SVG بديلة)
    └── icons/                الشعار
```

## التقنيات

HTML5, CSS3 (Grid/Flexbox + متغيرات CSS), JavaScript ES6+ فقط — بدون أي إطار عمل أو مكتبة خارجية (لا React، لا Bootstrap، لا jQuery). خط Cairo/Tajawal عبر Google Fonts.

## النشر على GitHub Pages

المشروع جاهز للنشر مباشرة: كل المسارات نسبية ولا يوجد أي اعتماد على خادم. يكفي رفع المستودع وتفعيل GitHub Pages من فرع `main` (المجلد الجذري).

## ملاحظات

- الصور المستخدمة (أغلفة الألعاب، الرسم التوضيحي) هي SVG بديلة (placeholders) تم إنشاؤها محليًا لأغراض العرض فقط.
- زر "شراء الآن" و"تسجيل الدخول" يفتحان نافذة منبثقة توضيحية فقط، دون أي عملية شراء أو تسجيل حقيقية.
- هذا نموذج قابل للتوسعة لاحقًا نحو منتج كامل (ربط ببوابة دفع، نظام حسابات، وقاعدة بيانات فعلية).
